# TouRend
Tour + Friend = TouRend - 여행 정보 제공, 동료 찾기 사이트


[회원 기능] -- 제작중

1) 회원가입
- ID(Email) 중복 확인, PW 일치 확인, PNum 인증 확인
- 유효성 검사

2) 회원수정
- Form - Username, PW, PW2, PNum(인증), Profile Photo

3) 비밀번호 찾기
ID(Email) 임시 패스워드 → Change Password

4) 로그인
ID(Email), PW 일치 → Set Session → Main Page

5) 로그아웃
Close Session

6) 카카오 및 구글 계정 로그인
- API 활용
